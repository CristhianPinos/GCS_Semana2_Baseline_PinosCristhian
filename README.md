# StockLight

## Objetivo del Proyecto
StockLight es una aplicación de línea de comandos (CLI) diseñada para gestionar el inventario de una pequeña tienda minorista. Permite registrar entradas y salidas de productos y mantener un control de stock en tiempo real mediante almacenamiento local. El objetivo principal de este repositorio es demostrar la gestión de configuración de software (GCS).

## Estructura del Repositorio
/docs -> Documentación de requisitos (SRS) y diseño (SDD)
/src -> Código fuente
/tests -> Casos de prueba
/config -> Archivos de configuración de ejemplo
/scripts -> Scripts auxiliares

## Cómo ejecutar el proyecto
Ejecutar el archivo main.py desde la carpeta src.

## Cómo crear la lìnea base (Baseline)
La línea base se crea mediante un tag Git (v1.0) y un Release asociado.
